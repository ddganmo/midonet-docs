 Transport Layer Port Range

The port range is a JSON object defining the boundaries of the port number interval to match. The start boundary must be smaller than the end boundary.

{ "start": 80, "end": 400 }

The range may be open-ended where, one of the range boundaries but not both may be missing.

{ "start": 80 }
{ "end": 400 }
