MidoNetでは、既存の物理ネットワークの上位のオーバーレイネットワークを運用します。 MidoNetエージェントはGW（ゲートウェイ）ノードとコンピュートホストのネットワークエッジのソフトウェアで実行します。

MidoNetエージェントに加え、ステイト管理システムによって分散されたエージェントそれぞれとも連係させます。一元管理のために、MidoNetではレストフルAPIサーバーも提供されます。
